# Bloom-Healthcare - Healthcare Data Analysis Project(Appclick Project)
Bloomhealth is a beginner-friendly healthcare data analysis project that explores patient demographics, hospital admissions, diagnoses, and medical billing data using SQL and Power BI. The project demonstrates data cleaning, querying, visualization, and insight generation.

Project Objectives
Analyze patient distribution by gender and state
Identify the most common medical diagnoses
Track hospital admissions over time
Analyze treatment costs and billing patterns
Identify high-cost medical cases
Build an interactive Power BI dashboard

Datasets Used
The project uses three datasets:
Patients: demographic information such as age, gender, and state
Admissions: hospital visits, diagnoses, and dates
Billing: treatment costs and insurance types

Tools & Technologies
SQL (MySQL)
Power BI
GitHub

Analysis & Dashboard
The Power BI dashboard includes:
KPI cards (Total Patients, Total Admissions, Total Billing)
Patients by Gender
Patients by State
Most Common Diagnoses
Monthly Hospital Admissions
Cost by Diagnosis
Top 10 Highest Medical Bills

Key Insights
Female patients are same as male patients
Malaria is the most common diagnosis
Patient admissions vary by month
Certain diagnoses account for higher treatment costs
